# decision-tree-random-forest
# Decision Tree and Random Forest

## Objective

To learn tree-based models for classification.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Dataset

Titanic Dataset

## Steps Performed

1. Loaded and preprocessed the dataset
2. Trained a Decision Tree model
3. Controlled overfitting using max_depth
4. Trained a Random Forest model
5. Compared model accuracy
6. Analyzed feature importance
7. Performed cross-validation

## Results

* Random Forest gave better accuracy than Decision Tree
* Sex and Pclass were important features
* Cross-validation showed stable performance

## Conclusion

Tree-based models are powerful for classification problems, and Random Forest generally performs better because it combines multiple trees.
